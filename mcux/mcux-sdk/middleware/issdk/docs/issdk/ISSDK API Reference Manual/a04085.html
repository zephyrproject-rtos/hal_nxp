IoT Sensing Software Development Kit
ISSDK Low Level Interface

ISSDK support direct low level interface to the sensor and register I2C or SPI interfaces.

Register I/O Interface

Function Description
Register_I2C_Write The low lever interface function to write to a sensor register over I2C
Register_I2C_Read The low lever interface function to read a sensor register over I2C
Register_SPI_Write The low lever interface function to write to a sensor register over SPI
Register_SPI_Read The low lever interface function to write to a sensor register over SPI

Sensor I/O Interface

Function Description
Sensor_I2C_Write The low lever interface function to write register data to a sensor over I2C
Sensor_I2C_Read The low lever interface function to read register data from a sensor over I2C
Sensor_SPI_Write The low lever interface function to write register data to a sensor register over SPI
Sensor_SPI_Read The low lever interface function to read register data from a sensor over SPI
