Detailed Description

Defines valid options to be used by mcuxClCss_HashOption_t.

Macro Definition Documentation

◆ MCUXCLCSS_HASH_INIT_ENABLE

#define MCUXCLCSS_HASH_INIT_ENABLE

Set this option at mcuxClCss_HashOption_t.hashini to initialize the hash.

Examples
hash_sha224_one_block.c, hash_sha256_one_block.c, hash_sha256_sha_direct_one_block.c, hash_sha384_one_block.c, hash_sha384_sha_direct_one_block.c, and hash_sha512_one_block.c.
