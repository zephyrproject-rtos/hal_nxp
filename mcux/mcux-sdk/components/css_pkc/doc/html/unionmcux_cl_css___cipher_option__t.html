Detailed Description

Command option bit field for mcuxClCss_Cipher_Async.

Bit field to configure mcuxClCss_Cipher_Async. See MCUXCLCSS_CIPHER_ for possible options.

Examples
cipher_aes128_cbc_encrypt.c, and cipher_aes128_ecb_encrypt.c.
