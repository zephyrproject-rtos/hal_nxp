# TSI_V6

  - Add TSI DPD mode support.
  - Add API TSI_EnableOutOfRangeDmaTransferOnly enable DMA transfer only when out of range.

  - Fixed incorrectly using the ALL_FLAGS_MASK when modifying the GENCS register issue
